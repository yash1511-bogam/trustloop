## Recommended Setup Order

1. Complete the [Getting Started](/docs/guides/getting-started) guide.
2. Resolve any authentication issues using [Sign In and Access](/docs/guides/identity-and-access).
3. Connect your first alert source via [Integrations](/docs/guides/integrations-and-ingestion).
4. Run a test incident following [Incident Operations](/docs/guides/incident-operations).
5. Review [Workspace Administration](/docs/guides/workspace-administration) before onboarding your team.
6. Familiarize yourself with [Plans, Billing, and Alerts](/docs/guides/billing-and-automation) to understand quotas and escalation policies.

## Platform Overview

TrustLoop provides a unified incident operations platform purpose-built for AI software companies:

- Ingest incidents from monitoring tools, webhooks, Slack, or manual entry.
- Assign ownership and maintain a single, auditable timeline of actions.
- Run AI-powered triage and generate customer-safe communication drafts.
- Publish status updates to a customer-facing status page.
- Manage workspace configuration, team access, API keys, notifications, and billing from a centralized settings interface.
