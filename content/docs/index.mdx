# Docs Overview

TrustLoop documentation is structured for fast onboarding, day-2 operations, and auditable communication.

We use a Diataxis-style content split so each page is intentionally one of: onboarding, how-to, reference, or explanation<sup id="ref-diataxis-overview"><a href="#src-diataxis-overview">[1]</a></sup>.

## Quick Paths

- [Getting Started](/docs/guides/getting-started)
- [Documentation Style and Citations](/docs/guides/documentation-style-and-citations)
- [Navigation and Backlinks](/docs/guides/navigation-and-backlinks)

## Documentation Guarantees

- Clear user intent by page type (learn, do, lookup, understand).
- Evidence-backed statements in policy pages.
- Internal links that keep readers in context before sending them out to external sources.
- One-click way back to docs home and product home from every major page.
